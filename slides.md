## 1. 工作流程标准化

提高效率 / 减少误解 / 确保一致性

<div class="text-xs" >

### 1.1 关键步骤

1. 需求分析 - PM
2. 研究和灵感收集 - ALL
3. 草图 / 原型设计 - PM + UI
4. 视觉设计 - UI
5. 前端开发 - FE
6. 测试 + 可用性优化 + ... - ALL

### 1.2 使用统一的工具

1. 通用工具 (slack / zoom / ...)
2. 开发工具 (github .. )
3. 设计工具 (figma, MasterGo ...)
4. 项目管理工具 (? 暂时没想法，似乎也不是瓶颈)

### 1.3 建立设计体系 （需要设计师来制定）
   1. logo / slogan 规范
   2. 创建组件库，定义颜色、字体、间距等设计规范
